# Data Engineering Portfolio 🚀
Este repositorio muestra ejemplos **end-to-end** de ingeniería de datos:
- Ingesta de datos desde APIs o datasets públicos
- Transformaciones y modelado con **dbt**
- Orquestación con **Airflow / Prefect**
- Analítica y visualización final
## Estructura
ingestion/         → Scripts de descarga y carga inicial  
transformations/   → Modelos dbt (dimensiones, hechos, snapshots)  
orchestration/     → DAGs o flujos de ejecución  
analytics/         → Notebooks o dashboards  
utils/             → Funciones auxiliares
